## **PHYSICAL THERAPY MODIFIER CHEAT SHEET**

### **Core Therapy Modifiers**

| Modifier | Meaning | When to Use |
| ----- | ----- | ----- |
| GP | PT services under a plan of care | Add to all PT CPT codes (e.g., 97110, 97140\) |
| KX | Exceeds therapy threshold, still medically necessary | Medicare – justify continued need |
| 59 | Distinct procedural service | When two normally bundled codes are separate |
| XE | Separate encounter | Same day, different session/time |
| XS | Separate structure | Same day, different body part |
| XP | Separate practitioner | Different provider (e.g., PT vs. OT) |
| XU | Unusual non-overlapping service | Distinct service not fitting above categories |

---

### **Assistant Modifiers**

| Modifier | Meaning | When to Use |
| ----- | ----- | ----- |
| CQ | Service provided in whole/part by PTA | Required for Medicare (triggers 85% payment) |
| CO | Service provided in whole/part by COTA | Required for OT claims under Medicare |

---

### **Coverage & Liability Modifiers**

| Modifier | Meaning | When to Use |
| ----- | ----- | ----- |
| GA | ABN issued (expected denial) | When patient signed ABN before service |
| GY | Statutorily non-covered service | For maintenance/wellness not covered by Medicare |
| GZ | Expected denial, no ABN signed | Used when no ABN obtained |
| 22 | Increased complexity/time | Rare – document justification |
| 76 | Repeat procedure, same provider | Same day, repeated service |
| 77 | Repeat procedure, different provider | Same day, different clinician |

### **Common Examples**

* 97110 \+ 97140 (different areas): 97140-GP-59 or 97140-GP-XS

* Medicare patient \> therapy cap: Add KX

* PTA provided part of session: Add CQ

* Maintenance exercise: Add GY
